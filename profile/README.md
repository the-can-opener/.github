# CAN Opener

**Turn any vehicle into a smart, connected system.**

### Welcome to the official CAN Opener GitHub

Modern vehicles are becoming software-defined, but most of the tooling around them is still closed, fragmented, or locked to specific platforms. CAN Opener is our attempt to change that.

We're building an open automotive software stack that makes vehicles easier to understand, extend, and program, from low-level CAN signals to mobile dashboards, cloud sync, widgets, and eventually full vehicle apps.

## Ecosystem

- **CAN Opener Hardware**: ESP32-based BLE adapter for reading and streaming vehicle CAN data through the OBD-II port.
- **can-opener-js**: TypeScript vehicle signal library for profiles, DBC decoding, queries, actions, and reactive vehicle state.
- **Nexus App**: React Native / Expo mobile app for dashboards, body controls, BLE pairing, profiles, and developer tools.
- **Cloud & Widgets**: Profile sharing, custom widgets, scripts, analytics, and future vehicle app infrastructure.
